Name

KXL_DeleteWindow

Synopsis
KXL_Delete -- Shut down KXL.

Description
void KXL_DeleteWindow(void);

Arguments

Nothing.


Return Value

Nothing.


Exsample
#include <KXL.h>

int main(void)
{
  KXL_CreateWindow(100, 100, "kxl.org", 0);
  getchar();
  KXL_DeleteWindow();
  return 0;
}
